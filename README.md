## 📖 About project

This project focuses on predicting house and apartment prices in Belgium using real market data. It collects property listings through web scraping, processes and analyzes the data, and applies data science and machine learning techniques to generate price predictions. The pipeline covers the full workflow – from raw data collection and cleaning to exploratory data analysis and model evaluation – providing practical insights into the Belgian real sestate market.

The project was developed as a learning initiative during our Data Science & AI school program. It was built collaboratively in a team with friends, giving us hands-on experience in teamwork, data engineering, model development, and real-world problem solving.

## 🚀 Getting started

This program has no GUI. This program also has no prepared running commands to work with. In future we expect updates with UI so everyone who wants to use our project can use it. You can get project on your PC just by downloading it. Running functions are in **main.py**:

- update_links(): scraping web and updating all links with data.
- update_dataset(): scraping links from previous function and updating **dataset.csv**.
- trim_data(): cleaning dataset from not relevant values. Not fully finished.
